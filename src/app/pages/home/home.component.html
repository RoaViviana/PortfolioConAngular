Este es mi Portfolio. Es el proyecto final del curso #YoProgramo de Argentina Programa. En el cual he aplicado todos los conocimientos adquiridos. Comenzando por el lado del FrontEnd con el maquetado del sitio utilizando HTML, CSS y JavaScript, incorporando luego Angular como Framework y TypeScript como lenguaje. Dando uso a muchas de sus herramientas como la creación de componentes, templates, servicios, módulos, directivas, databinding, interfaces, routing, lazyload, guards, pipes, formularios Reactivos, sesión de usuario, uso de Bootstrap como Framework Styles. Para la realización del BackEnd se empleó Spring Boot como framework + JPA + Hibernate con Java como lenguaje y MySQL como gestor de base de datos. Se crearon las API necesarias para responder a todas las peticiones del FrontEnd. Se utilizó arquitectura multicapa, Controller, Model, Repository, Service, Annotations, inyección de dependencias, Autenticación con JWT, seguridad en rutas con gestión de Roles y upload de imágenes. Sinceramente la realización de este curso ha sido una experiencia muy positiva y motivadora, agradezco profundamente a todas aquellas personas que hicieron posible esta capacitación, han puesto a nuestro alcance un mundo de nuevas oportunidades con todo lo que ello significa para muchos de nosotros. ¡Gracias de corazón!.
